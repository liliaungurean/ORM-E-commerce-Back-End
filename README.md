## Installation
Install the following; 

npm install

npm install mysql2

npm install sequelize

npm install dotenv

## Usage

Run the following command at the root of your project and answer the prompted questions:

mysql -u root -p

    Enter password

source db/schema.sql

quit

npm run seed

npm start
